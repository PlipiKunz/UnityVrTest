
Dependencies
	Features
		Uses the Engineering and VR Features

	Packages
		Uses the "XR Interaction Toolkit" https://docs.unity3d.com/Packages/com.unity.xr.interaction.toolkit@2.2/manual/index.html, it pulls in starter assets and the XR Device Simulator which are required dependencies.

		Uses the "OpenXR Plugin", installs the Controller sample, which is anpther dependency.

		Also pulls in the "MockHMD XR Plugin", and "XR Plugin Management" for vr systems, among others.
